# RISC-V-Emulator

# Implemented instructions:

### Arithmetic

- add
- addi
- sub

### Bitwise Logic

- and
- andi
- or
- ori
- xor

### Load Immediate

- li
- lui
- auipc

### Branch

- beq
- bne

## Jump and Function

- j
- jal
- jalr


### Load and Store

- lw
- sw


### Known bugs:
- If the editor wraps the text, blocks not parsed well
- \n are not recognized
- offset not working properly (should be multiple of 4)
